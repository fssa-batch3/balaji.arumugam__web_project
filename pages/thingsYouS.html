Let's discuss about the folders first.

What is a folder?

Let's take an example as tree , In a tree there will be a lot of branches, in a specific branch there will be a lot of sub-branches. This is a great example of folders.

Likewise,A folder can also contain other folders.(i.e:Binary tree data structure)
A folder is a container for storing files, such as documents, spreadsheets, applications, etc.

What is Zip folder?

A zip folder is nothing but it have some restriction to do something with it.If you try to edit the files inside a folder you can't do,because the ZIP means it is locked..

Restrictions:You cannot edit the file without unzip it.

How to Unzip the folder?

If you right click it then You will find "Extract all" option.Select it.Then you will ask for location where to Extract.Choose your location.

Then,You are good to go...Yeah!!You can edit the file.

How to zip the folder?

You can right click the folder and click "Send to" then click "Compressed Zipped folder"
